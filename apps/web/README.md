# IA-ContactCenter-Multichannel-MVP
Repositorio del estado actual del MVP de la plataforma de contact center + CRM + IA. Incluye frontend, backend, integraciones y estructura de datos base, y servirá como punto de partida y referencia para la evolución hacia una arquitectura modular con Core Platform, SaaS Control Plane y Domain Apps.

# MVP Contact Center + CRM + IA

Este repositorio conserva el estado actual del MVP funcional de la plataforma de contact center + CRM + IA.

Su propósito es:
- preservar una base estable del MVP actual,
- servir como referencia para el refactor a arquitectura modular,
- y facilitar la migración hacia una plataforma SaaS multitenant reutilizable.

## Alcance actual
- gestión de campañas
- leads
- conversaciones
- handoff humano
- analítica básica
- canales base
- flujos con IA
- integraciones del MVP

## Nota
Este repositorio representa una fotografía técnica del MVP previo a la separación en:
- Core Platform
- SaaS Control Plane
- Contact Center / CRM / AI Domain
